# ⚡ Calculadora de Consumo de Energia

Desenvolvi esse projeto com o objetivo de calcular o consumo mensal de energia de um aparelho elétrico, de forma bem objetiva  e prática.

## 🧠 Como funciona

O programa pede ao usuário:
- Nome do aparelho
- Potência em watts (W)
- Tempo de uso diário (em horas)

Com base nesses dados, ele calcula o consumo mensal utilizando a fórmula:

(consumo = potência × horas × 30) / 1000

## 💰 Cálculo do custo

O sistema calcula o consumo e o valor mensal a ser pago, considerando R$ 0,75 por kWh.

## 💻 Tecnologias utilizadas

- Python
- VS Code

## ▶️ Como executar

Abra o terminal e execute:

python app.py

## 📌 Observação

Os valores devem ser digitados apenas em números (ex: 45 e não 45W).
